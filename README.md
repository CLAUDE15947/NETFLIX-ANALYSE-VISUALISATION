# Contexte :
Analyse d’un grand dataset Netflix (films et séries).
Besoin de visualiser et comprendre rapidement le catalogue (types, pays, genres, durée, ratings).


# Objectif du projet :
Nettoyer et préparer les données (durée, dates, genres, etc.).
Créer un dashboard interactif avec Dash et Plotly pour explorer le catalogue.
Implémenter un modèle de classification utilisant TF-IDF pour prédire “Movie” ou “TV Show”.
Fournir un exemple concret d’application de la Data Science à des données réelles.

# Analyse du Catalogue Netflix + Dashboard Interactif + Modèle de Prédiction

Ce projet analyse le catalogue Netflix grâce à :
- un **nettoyage approfondi du dataset**
- une **analyse exploratoire (EDA)**
- un **dashboard interactif (Plotly + Dash)**
- un **modèle de prédiction utilisant TF-IDF + Régression Logistique** permettant de déterminer si une description correspond à un *Film* ou une *Série*.
